Project Euler in R

Project Euler is consistently updated set of math and computer programing based problems meant to help improve working knowledge of code and build problem solving skills. I have been going through these problems in my spare time both to improve my own programing, as well as to document the solutions to the problem set in the R programming language.

My solutions are often clunky and imperfect but they do produce the right answers! hopefully they can help to provide some amount of assistance to others. You can look at my current solutions on GitHub.
